## 🔍 Troubleshooting common errors

If the installation fails, try these methods to fix the issue.

### Clear previous installation files
An old, corrupted version of the software often prevents a new installation. 

1.  Press the Windows key and type "Add or remove programs."
2.  Search for "Lucidchart" in the list.
3.  If you find it, click "Uninstall."
4.  Restart your computer before you initiate the new installation.

### Run as administrator
Permissions often block the installer. 

1.  Right-click on the installation file you downloaded.
2.  Select "Run as administrator" from the menu.
3.  Proceed with the steps as normal.

### Check your antivirus settings
Antivirus software may flag the installer by mistake. Temporarily disable your antivirus protection, run the Lucidchart installer, and then turn your protection back on. Refer to your antivirus software instructions for guidance on how to whitelist an application.

### Update your graphics drivers
The Lucidchart application relies on modern graphics drivers to render diagrams correctly.

1.  Right-click the Start button and select "Device Manager."
2.  Expand the "Display adapters" section.
3.  Right-click your graphics card.
4.  Select "Update driver."
5.  Choose "Search automatically for drivers."

### Ensure Windows is updated
Outdated Windows versions lack the necessary system files to run the latest software. 

1.  Open "Settings."
2.  Select "Windows Update."
3.  Click "Check for updates."
4.  Install all pending updates and restart your computer.

## 📈 Frequently asked questions

**Why does the setup hang at 99%?**
This usually indicates an issue with a background process that is already using a Lucidchart file. Open Task Manager by pressing Ctrl + Shift + Esc, look for any Lucidchart processes under the "Details" tab, and end those tasks.

**Does the desktop version work offline?**
The desktop application lets you view your diagrams offline, but you need an internet connection to sync your changes to the cloud.

**Can I use multiple accounts?**
The software supports one account per session. You must sign out of the current account before logging in with a different set of credentials.

**Where does the application store my diagrams?**
Your diagrams live in the cloud. The desktop version acts as a portal to your online account.
